# 3D Portfolio

This is an interactive **3D Portfolio** designed using **React**, **Three.js**, and **React-Spring** to create a visually dynamic and engaging personal portfolio. The website incorporates 3D animations, interactive components, and smooth transitions to showcase your skills and projects. The goal of this project is to create a modern, creative, and eye-catching portfolio using advanced web technologies.

You can view the live version of the project [here](https://abressiddique.github.io/3d-portfolio-site/).

## Features

- **3D Animations**: Utilizes **Three.js** and **React-Spring** to bring 3D graphics and smooth animations to the portfolio.
- **Interactive Elements**: Clickable components that lead to your projects or sections, with parallax effects and smooth transitions.
- **Responsive Design**: Optimized for both mobile and desktop viewing.
- **Email Integration**: Integrated with **EmailJS** for sending messages directly from the website.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Three.js**: 3D graphics library for creating interactive 3D animations and scenes.
- **React-Spring**: React library for animations and transitions.
- **React Router**: For handling navigation between different sections of the portfolio.
- **TailwindCSS**: A utility-first CSS framework for fast styling and customizations.
- **EmailJS**: Service to send emails directly from the client-side.

## Prerequisites

Ensure you have **Node.js** and **npm** (or **yarn**) installed on your system.

- Install [Node.js](https://nodejs.org/) (which comes with npm).
- Ensure you have npm version 7 or higher.
